ESP32 AI 자율주행 RC카
[ 소프트웨어 실습 파일 ]

아래 순서대로 실습을 진행합니다. 링크를 클릭하여 .ipynb 파일을 다운로드하세요.

  1. 텔넷 키보드 주행 (기본 제어 확인) ※ 카메라 영상이 나오는지 확인하고, 키보드로 RC카를 직접 조종하여 회로 연결 상태를 점검합니다.
  2. 주행 데이터 수집 (Data Collection) ※ 직접 운전하며 학습용 데이터를 수집합니다. 'N'키를 눌러 새 폴더를 만든 후 주행하세요.
  3. 티처블 머신 모델 테스트 (PC용) ※ 구글 Teachable Machine에서 만든 keras_model.h5 파일을 PC에 저장된 이미지로 테스트해봅니다.
  4. 티처블 머신 실시간 자율주행 ※ 티처블 머신 모델을 RC카에 적용하여 바로 자율주행을 시도합니다. (데이터 수집 불필요)
  5. CNN 모델 직접 학습시키기 (Training) ※ 2번 단계에서 수집한 데이터를 불러와 직접 CNN 인공지능 모델을 학습시키고 model.h5를 만듭니다.
  6. CNN 모델 자율주행 (기본 주행) ※ 직접 학습시킨 모델을 로드하여 RC카가 스스로 운전하게 합니다.
  7. MobileNetV2 전이 학습 (심화 과정) ※ 더 성능이 좋은 MobileNetV2 모델을 사용하여 학습 성능을 높여봅니다.
  8. 8. 멀티스레드 고성능 자율주행 (최적화) ※ 영상 수신과 AI 판단을 동시에 처리(멀티스레드)하여 반응 속도를 획기적으로 높인 최종 코드입니다.
